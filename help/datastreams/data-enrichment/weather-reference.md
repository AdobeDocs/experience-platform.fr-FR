# Mappages des champs de données météo

Adobe s’est associé à [!DNL [The Weather Company]](https://www.ibm.com/weather) pour ajouter la météo des États-Unis aux données collectées par les trains de données. Vous pouvez utiliser ces données pour les analyses, le ciblage et la création de segments dans Experience Platform.

Cette page répertorie tous les champs disponibles que vous pouvez utiliser pour enrichir les données de votre audience.

Les champs sont répartis en trois groupes différents qui s’alignent sur les groupes de champs.

* [**[!UICONTROL Current Weather]**](#current-weather) : conditions météorologiques actuelles de l’utilisateur, en fonction de son emplacement. Ces données affichent la température actuelle, les précipitations, la couverture nuageuse, etc.
* [**[!UICONTROL Forecasted Weather]**](#forecast) : la prévision inclut la prévision sur 1, 2, 3, 5, 7 et 10 jours pour l’emplacement de l’utilisateur.
* [**[!UICONTROL Triggers]**](#triggers) : les déclencheurs sont des combinaisons spécifiques qui correspondent à différentes conditions météorologiques sémantiques. Il existe trois types de déclencheurs météorologiques :

   * **[!UICONTROL Relative Triggers]** : conditions significatives d’un point de vue sémantique, telles qu’un temps froid ou pluvieux. Leurs définitions peuvent varier selon les climats.
   * **[!UICONTROL Product Triggers]** : conditions qui conduiraient à l&#39;achat de différents types de produits. Par exemple, les prévisions de périodes de froid augmentent la probabilité d’achat de manteaux.
   * **[!UICONTROL Severe Weather Triggers]** : avertissements météorologiques violents, comme les avertissements de tempête hivernale ou d&#39;ouragan.

## Déclencheurs {#triggers}

| Champ | Description | Chemin XDM |
| --- | ---- | --- |
| [!UICONTROL Product Triggers] | Ils indiquent les conditions qui conviennent pour stimuler les ventes de certaines catégories de produits.  Ils sont mappés à des nombres entiers. Vous pouvez obtenir une liste complète auprès d’IBM. | `weather.productTriggers` |
| [!UICONTROL Relative Triggers] | Conditions relatives basées sur la perception humaine. Telles que le chaud ou le froid. Ils sont mappés à des nombres entiers. Vous pouvez obtenir une liste complète auprès d’IBM. | `weather.relativeTriggers` |
| [!UICONTROL Severe Weather Triggers] | Indique diverses conditions météorologiques extrêmes, telles que les ouragans ou les pluies diluviennes. | `weather.severeTriggers` |

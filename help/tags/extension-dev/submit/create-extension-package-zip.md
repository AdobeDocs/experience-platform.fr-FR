# Créer un fichier zip de package d’extension

Pour préparer un package d’extension à charger dans Adobe Experience Platform, consultez la documentation API pour l’outil de ligne de commande [@adobe/reactor-packager](https://www.npmjs.com/package/@adobe/reactor-packager). Il génère un fichier zip préparé au format attendu pour le [chargement d’une extension](./upload-and-test.md).

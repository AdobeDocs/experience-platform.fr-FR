# Types d’actions

Cette page répertorie toutes les actions prises en charge dans une règle de balise. Pour créer ou modifier les actions dans une règle de balise :

1. Connectez-vous à [experience.adobe.com](https://experience.adobe.com?lang=fr) à l’aide de vos informations d’identification Adobe ID.
1. Accédez à **[!UICONTROL Data Collection]** > **[!UICONTROL Tags]**.
1. Sélectionnez la propriété de balise de votre choix.
1. Accédez à **[!UICONTROL Rules]**, puis sélectionnez la règle de votre choix.
1. Sous [!UICONTROL Actions], sélectionnez une action existante ou créez-en une.
1. Définissez le champ déroulant [!UICONTROL Extension] sur **[!UICONTROL Adobe Experience Platform Web SDK]**, puis sélectionnez le [!UICONTROL Action Type] souhaité.

L’extension de balise Web SDK propose les actions suivantes :

* [**[!UICONTROL Apply propositions]**](apply-propositions.md) : propositions de rendu dans les applications monopages sans incrémenter de mesures.
* [**[!UICONTROL Apply response]**](apply-response.md) : permet d’effectuer une action en fonction d’une réponse d’Edge Network.
* [**[!UICONTROL Evaluate rulesets]**](evaluate-rulesets.md) : déclenchez manuellement une évaluation de jeu de règles.
* [**[!UICONTROL Get Media Analytics tracker]**](get-media-analytics-tracker.md) : exportez l’API Media héritée vers un objet de fenêtre.
* [**[!UICONTROL Redirect with identity]**](redirect-with-identity.md) : permet le partage d’un identifiant visiteur entre les domaines que vous possédez.
* [**[!UICONTROL Send event]**](send-event.md) : envoie les données d’événement à Edge Network.
* [**[!UICONTROL Send media event]**](send-media-event.md) : envoie les données de média à Edge Network.
* [**[!UICONTROL Set consent]**](set-consent.md) : permet de définir le consentement souhaité pour le visiteur.
* [**[!UICONTROL Update variable]**](update-variable.md) : permet de modifier un élément de données [variable](../data-element-types.md#variable).

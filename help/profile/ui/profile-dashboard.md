#  Tableau de bord des profils  {#profile-dashboard}

L’interface utilisateur de Adobe Experience Platform fournit un tableau de bord grâce auquel vous pouvez afficher des informations importantes sur vos données [!DNL Real-time Customer Profile], telles qu’elles sont capturées lors d’un instantané quotidien.

Pour obtenir des instructions détaillées sur l’accès et l’interaction avec le tableau de bord [!UICONTROL Profils] dans l’interface utilisateur, ainsi que pour en savoir plus sur les mesures disponibles affichées dans le tableau de bord, consultez le guide [[!UICONTROL Profils] du tableau de bord](../../dashboards/guides/profiles.md).

Pour un aperçu de toutes les fonctionnalités du tableau de bord dans l’interface utilisateur de l’Experience Platform, commencez par lire la [présentation des tableaux de bord](../../dashboards/home.md).

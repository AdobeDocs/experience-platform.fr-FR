# Présentation de Data Distiller

Data Distiller est une offre de package qui comprend un sous-ensemble des fonctionnalités de Adobe Experience Platform. Data Distiller vous permet d’effectuer une préparation des données post-ingestion (comme le nettoyage, la mise en forme et la manipulation) pour Real-Time Customer Profile ou des cas d’utilisation analytiques en exécutant des requêtes par lots dans Query Service. Vous pouvez également [créer des modèles de données de rapport personnalisés ;](../../query-service/data-distiller/customizable-insights/reporting-insights-data-model.md) ou [utiliser SQL pour générer des graphiques pour vos tableaux de bord personnalisés ;](./customizable-insights/overview.md).

>[!NOTE]
>
>Votre utilisation de Data Distiller dépend de vos droits pour les applications basées sur Platform.

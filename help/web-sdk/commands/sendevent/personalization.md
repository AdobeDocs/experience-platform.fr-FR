# `personalization`

La variable `personalization` contient plusieurs propriétés qui vous permettent de personnaliser votre expérience utilisateur en fonction de l’utilisateur. Elle est particulièrement utile dans les implémentations d’Adobe Target ou de Adobe Journey Optimizer, car c’est la force motrice qui vous permet de personnaliser le contenu affiché par utilisateur.

Consultez la documentation sur la manière de [rendu du contenu personnalisé](../../personalization/rendering-personalization-content.md) pour plus d’informations.

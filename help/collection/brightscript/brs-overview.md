# Présentation de Brightscript

Le **SDK Roku Adobe Experience Platform** vous permet d’envoyer des données à un [flux de données](/help/datastreams/overview.md) à partir d’un appareil Roku à l’aide de Brightscript.

Le SDK et la documentation pour cette bibliothèque sont hébergés sur GitHub :

**[Adobe Experience Platform Roku SDK](https://github.com/adobe/aepsdk-roku)**

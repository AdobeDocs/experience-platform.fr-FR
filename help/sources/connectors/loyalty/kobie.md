# [!DNL Kobie]

[!DNL Kobie] fournit une plateforme SaaS composable que vous pouvez utiliser pour gérer l’ensemble du cycle de vie d’un programme de fidélité. Généralement utilisée par les équipes marketing, les spécialistes des données et les responsables de l’expérience client, la plateforme [!DNL Kobie] comprend une interface sans code que vous pouvez utiliser pour orchestrer des règles de fidélité complexes.

Vous pouvez utiliser les sources de [!DNL Kobie] du catalogue des sources Adobe Experience Platform pour ingérer des données de fidélité par lots et par flux à partir de votre compte [!DNL Kobie].

- [diffusion en continu [!DNL Kobie]](../../tutorials/ui/create/loyalty/kobie-streaming.md)

# [!DNL Salesforce] connecteur

Adobe Experience Platform allows data to be ingested from external sources while providing you with the ability to structure, label, and enhance incoming data using [!DNL Platform] services. Vous pouvez ingérer des données à partir de diverses sources, telles que des applications Adobe, des enregistrements basés sur le cloud, des bases de données, etc.

[!DNL Experience Platform] prend en charge l’assimilation de données à partir d’un système de gestion de la relation client tiers. La prise en charge des fournisseurs de gestion de la relation client comprend [!DNL Salesforce].

La documentation ci-dessous fournit des informations sur la façon de se connecter [!DNL Salesforce] à [!DNL Platform] l’aide des API ou de l’interface utilisateur :

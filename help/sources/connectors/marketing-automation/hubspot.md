# (Bêta) [!DNL HubSpot] Connecteur

>[!NOTE]
>Le [!DNL HubSpot] connecteur est en version bêta. Pour plus d’informations sur l’utilisation de connecteurs bêta, consultez l’aperçu [des](../../home.md#terms-and-conditions) sources.

Adobe Experience Platform allows data to be ingested from external sources while providing you with the ability to structure, label, and enhance incoming data using [!DNL Platform] services. Vous pouvez ingérer des données provenant de diverses sources telles que les applications Adobe, le stockage dans le cloud, des bases de données, etc.

[!DNL Experience Platform ]prend en charge l’ingestion de données provenant d’un système tiers d’automatisation du marketing. La prise en charge des fournisseurs d’automatisation de marketing inclut [!DNL HubSpot].

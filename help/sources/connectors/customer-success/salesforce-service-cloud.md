# (Bêta) [!DNL Salesforce Service Cloud] Connecteur

>[!NOTE]
>Le [!DNL Salesforce Service Cloud] connecteur est en version bêta. Pour plus d’informations sur l’utilisation de connecteurs bêta, consultez l’aperçu [des](../../home.md#terms-and-conditions) sources.

Adobe Experience Platform allows data to be ingested from external sources while providing you with the ability to structure, label, and enhance incoming data using [!DNL Platform] services. Vous pouvez ingérer des données à partir de diverses sources, telles que des applications Adobe, des enregistrements basés sur le cloud, des bases de données, etc.

[!DNL Experience Platform] prend en charge l’importation de données à partir d’un système de succès client tiers. La prise en charge des fournisseurs de succès client comprend [!DNL Salesforce Service Cloud].

La documentation ci-dessous fournit des informations sur la façon de se connecter [!DNL Salesforce Service Cloud] à [!DNL Platform] l’aide des API ou de l’interface utilisateur :

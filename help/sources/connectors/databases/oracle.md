# Connecteur Oracle

Adobe Experience Platform fournit une connectivité native pour les fournisseurs de base de données tels que Microsoft, MySQL et Azure. Vous pouvez importer vos données de ces systèmes dans Platform.

Différents types de bases de données tierces sont pris en charge, y compris les entrepôts de données relationnels, NoSQL ou Data Warehouse. Les fournisseurs de base de données prennent en charge Oracle.

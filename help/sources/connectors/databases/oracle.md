# (Bêta) Connecteur Oracle

>[!NOTE]
>Le connecteur Oracle est en version bêta. Pour plus d’informations sur l’utilisation de connecteurs bêta, consultez l’aperçu [des](../../home.md#terms-and-conditions) sources.

Adobe Experience Platform fournit une connectivité native aux fournisseurs de base de données tels que Microsoft, MySQL et Azure. Vous pouvez importer vos données de ces systèmes dans Platform.

Différents types de bases de données tierces sont pris en charge, y compris relationnelles, NoSQL ou les datas warehouse. Les fournisseurs de base de données prennent en charge Oracle.

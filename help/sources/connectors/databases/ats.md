# (Bêta) [!DNL Azure Table Storage] Connecteur

>[!NOTE]
>Le [!DNL Azure Table Storage] connecteur est en version bêta. Pour plus d’informations sur l’utilisation de connecteurs bêta, consultez l’aperçu [des](../../home.md#terms-and-conditions) sources.

Adobe Experience Platform allows data to be ingested from external sources while providing you with the ability to structure, label, and enhance incoming data using [!DNL Platform] services. Vous pouvez ingérer des données à partir de diverses sources, telles que des applications Adobe, des enregistrements basés sur le cloud, des bases de données, etc.

[!DNL Experience Platform] prend en charge l’importation de données à partir d’une base de données tierce. [!DNL Platform] peut se connecter à différents types de bases de données, tels que relational, NoSQL ou datas warehouse. Les fournisseurs de base de données sont pris en charge [!DNL Azure Table Storage].

La documentation ci-dessous fournit des informations sur la façon de se connecter [!DNL Azure Table Storage] à [!DNL Platform] l’aide des API ou de l’interface utilisateur :

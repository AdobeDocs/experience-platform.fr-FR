# (Bêta) Connecteur GreenPlum

>[!NOTE]
>Le connecteur GreenPlum est en version bêta. Pour plus d’informations sur l’utilisation de connecteurs bêta, consultez l’aperçu [des](../../home.md#terms-and-conditions) sources.

Adobe Experience Platform fournit une connectivité native aux fournisseurs de base de données tels que Microsoft, MySQL et Azure. Vous pouvez importer vos données de ces systèmes dans Platform.

Différents types de bases de données tierces sont pris en charge, y compris relationnelles, NoSQL ou les datas warehouse. La prise en charge des fournisseurs de base de données inclut GreenPlum.

La documentation ci-dessous fournit des informations sur la connexion de GreenPlum à Platform à l’aide d’API ou de l’interface utilisateur :

# (Bêta) [!DNL Couchbase] Connecteur

>[!NOTE]
>
>Le [!DNL Couchbase] connecteur est en version bêta. Pour plus d’informations sur l’utilisation de connecteurs bêta, consultez l’aperçu [des](../../home.md#terms-and-conditions) sources.

Adobe Experience Platform fournit une connectivité native pour les fournisseurs de base de données tels que [!DNL Microsoft], MySQL et [!DNL Azure], ce qui vous permet d&#39;apporter vos données à partir de ces systèmes. Différents types de bases de données tierces sont pris en charge, y compris les entrepôts de données relationnels, NoSQL ou Data Warehouse. La prise en charge des fournisseurs de base de données comprend [!DNL Couchbase].

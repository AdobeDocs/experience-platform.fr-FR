# (bêta) Connecteur PayPal

>[!NOTE]
>Le connecteur PayPal est en version bêta. Pour plus d’informations sur l’utilisation de connecteurs bêta, consultez l’aperçu [des](../../home.md#terms-and-conditions) sources.

L’Adobe Experience Platform permet l’assimilation de données à partir de sources externes tout en vous permettant de structurer, d’étiqueter et d’améliorer les données entrantes à l’aide des services Platform. Vous pouvez ingérer des données à partir de diverses sources, telles que des applications Adobe, des enregistrements basés sur le cloud, des bases de données, etc.

L’Experience Platform prend en charge l’importation de données à partir d’une application de paiement tierce. PayPal est l&#39;un des fournisseurs de paiements pris en charge.

La documentation ci-dessous fournit des informations sur la connexion de PayPal à Platform à l&#39;aide des API ou de l&#39;interface utilisateur :

# Connecteur FTP et SFTP

Adobe Experience Platform permet d’assimiler des données à partir de sources externes tout en vous permettant de structurer, d’étiqueter et d’améliorer les données entrantes à l’aide des services de plateforme. Vous pouvez assimiler des données à partir de diverses sources, telles que des applications Adobe, des  basés sur le cloud, des bases de données et bien d’autres.

Cloud  sources de peuvent importer vos propres données dans Platform sans avoir à les télécharger, les mettre en forme ou les télécharger. Les données insérées peuvent être formatées au format JSON XDM, parquet XDM ou délimitées. Chaque étape du processus est intégrée dans le flux de travaux Sources à l’aide de l’interface utilisateur. Les serveurs FTP et SFTP sont pris en charge pour les  de cloud  les fournisseurs de services de messagerie instantanée.

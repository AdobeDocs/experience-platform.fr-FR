# Jeux de données

Dans l’interface utilisateur d’Adobe Experience Platform, vous pouvez vue et gérer tous les jeux de données créés par votre organisation IMS, ainsi que en créer de nouveaux. Pour parcourir les jeux de données dans l’interface utilisateur, sélectionnez **[!UICONTROL Jeux de données]** dans le volet de navigation de gauche. The *[!UICONTROL Browse]* tab contains a list of datasets, including columns showing Name, Created (date and time), Source, Schema, and Last Batch Status, as well as the date and time the dataset was Last Updated.

Pour plus d’informations sur les jeux de données dans la plateforme d’expérience, commencez par lire l’aperçu [des jeux de](../../catalog/datasets/overview.md)données.

# Créateur de segments dans Real-Time Customer Data Platform

Basé sur Adobe Experience Platform, [!DNL Adobe Real-Time Customer Data Platform] peut utiliser les fonctionnalités complètes du créateur de segments qui font partie de [!DNL Experience Platform]. Dans l’interface utilisateur du créateur de segments, vous pouvez créer de nouvelles audiences, accéder à des audiences existantes et modifier des audiences existantes.

Pour obtenir un guide plus détaillé sur le créateur de segments, consultez le [guide d’utilisation du créateur de segments](../../segmentation/ui/segment-builder.md).

# Sources dans Real-time Customer Data Platform

[!DNL Real-Time Customer Data Platform] permet d’ingérer des données à partir de sources externes tout en vous permettant de structurer, de libeller et d’améliorer les données entrantes à l’aide des services [!DNL Platform]. Vous pouvez envoyer des données à partir de différentes sources, notamment les solutions Adobe, le stockage dans le cloud, des logiciels tiers et la gestion de la relation client.

Pour une présentation plus détaillée des sources, y compris une liste des sources disponibles et des liens vers les tutoriels, veuillez lire la [documentation sur les sources](../../sources/home.md).

# Connexion des clients au service de Requête

Cette section explique comment se connecter à Requête Service à partir de diverses applications clientes de bureau et comment vérifier ces connexions. Requête Service utilise le protocole Postgres, de sorte que les instructions de cette section expliquent comment utiliser les outils Postgres et les pilotes pour se connecter et écrire des requêtes.

Des instructions sont fournies aux clients suivants :

- [Aqua Data Studio](./aqua-data-studio.md)
- [Recherche](./looker.md)
- [Postico (Mac)](./postico.md)
- [Power BI (PC)](./power-bi.md)
- [PSQL](./psql.md)
- [RStudio](./rstudio.md)
- [Tableau](./tableau.md)
